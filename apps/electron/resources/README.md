# アプリケーションアイコン

Electronアプリのアイコンファイルを配置するディレクトリです。

## 配置済みファイル

- `icon.icns` - Mac用アイコン
- `icon.ico` - Windows用アイコン
- `icon.png` - 共通PNG画像

## アイコンの更新方法

1. 1024x1024pxの正方形PNG画像を用意
2. オンラインツールで変換:
   - Mac (.icns): https://cloudconvert.com/png-to-icns
   - Windows (.ico): https://cloudconvert.com/png-to-ico
